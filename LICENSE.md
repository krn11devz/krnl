### ❌ Issue 3: Scripts Crash Immediately After Execution

**Cause:** Script incompatibility or outdated KRNL build
**Fix:**
- Update to the latest version
- Check the script's UNC requirements against KRNL's current UNC level
- Try clearing the script cache:

**Fix:**

3. Re-launch the app
